# lnwsj · Apps Showcase

🎨 รวมผลงานจาก AI — 6 apps แบ่งเป็น 2 หมวด

## 🌐 Web Apps (live demo)
- **👗 [FitCheck Studio](https://codehub.sj88ai.com/fitcheck/)** — 3D dress-up game with AI Fashion Advisor
- **🎤 [Karaoke Subtitle Studio](https://codehub.sj88ai.com/karaoke-studio/)** — lyric video maker ในเบราว์เซอร์

## 🛠 Desktop Tools (in-browser port + Python source)
- **🎬 [MiniCut (lnwSJCut)](https://github.com/lnwsj/lnwSJCut)** — Video editor (Flet + ffmpeg)
- **🖼️ [PNG→JPG Converter](https://github.com/lnwsj/png_to_jpg_gui)** — batch image converter
- **✂️ [Background Remover](https://github.com/lnwsj/remove-bg-free)** — AI cutout (rembg)
- **🏷️ [Batch Rename by Template](https://github.com/lnwsj/rename_by_template_gui)** — file renamer (natural sort)

## ▶ How to use this showcase

เปิด `https://lnwsj.github.io/demo-sj88ai/` แล้ว:
1. **Web Apps** — กด "Launch Live Demo" เพื่อเปิดเต็มจอ
2. **Desktop Tools** — กด "▶ Web Demo" เพื่อลองใช้ในเบราว์เซอร์, หรือกด "📂 Run on desktop" เพื่อดูคำสั่ง install

ทุก app มี working demo — ไม่ใช่แค่ screenshot

## 🧰 Stack
- Single-file HTML (no build)
- Inline CSS + Vanilla JS
- CDN libs: JSZip, @imgly/background-removal (ESM imports)
